# Investie - Financial Investment Aid

투자 도우미 웹 애플리케이션입니다.

## 기능

- ✅ 투자 수익률 계산기
- ✅ 포트폴리오 추적 및 관리
- ✅ 주식 가격 조회 (미국 주식)
- ✅ 환율 조회 및 변환
- ✅ 로컬 스토리지에 포트폴리오 저장

## 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. 원하는 탭 선택:
   - **투자 계산기**: 투자 금액과 현재 가치를 입력하여 수익률 계산
   - **포트폴리오**: 보유 주식을 추가하고 손익 추적
   - **주식 조회**: 미국 주식 심볼로 현재 가격 조회
   - **환율 조회**: 실시간 환율 정보 조회 및 변환

## 기술 스택

- HTML/CSS/JavaScript
- ExchangeRate-API (환율)
- Yahoo Finance API (주식 가격, 프록시 사용)
- Local Storage
